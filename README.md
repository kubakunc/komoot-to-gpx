# komoot-to-gpx

Tools to export your private Komoot tours as GPX files.

## Subprojects

- **[`web/`](./web)** — the original PWA (SvelteKit + Node, self-hosted via Docker + Cloudflare Tunnel).
- **[`mobile/`](./mobile)** — native Android app for Google Play, branded as **Export GPX for Komoot** by **VeloLogic Labs** (Capacitor + SvelteKit SPA). _Work in progress._

See `docs/superpowers/specs/` for design documents and `docs/superpowers/plans/` for implementation plans.
